Tweets for the 2015 Barcelona City Council election

JSON file of the tweets with the following fields:

    id: ID of the tweet
    timestamp_ms: Timestamp in millisecs
    text: Text of the tweet


Some tweets can include some of the following fields:

    entities_hashtags_text: Array of hashtags 
    entities_media_expanded_url: Array of media URLs (photo/video)
    entities_urls_expanded_url: Array of other URLs (websites)
    retweeted_status_id: If the tweet is a retweet (text starts with "RT @"), ID of the original tweet is being retweeted
